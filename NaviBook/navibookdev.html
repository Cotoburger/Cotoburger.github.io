INFO

Our development team also provides a convenient set of SDKs for software development. The App SDK provides a ready-made board with existing internal connections and the necessary checks, you only need to connect the SDK to the laptop board, draw the application icon and the logic itself. The SDK has a copy of the display for your convenience, you only need to display the image on it, no additional checks on your part are required.

FEATURES

The NaviBook system comes with pre-built features to help developers quickly prototype their ideas. You get seamless integration with the onboard hardware and advanced debugging options out of the box.

GET STARTED

To get started, simply download the SDK and follow the integration guide provided in our documentation. You can also access community resources and tutorials to help you with your project.
